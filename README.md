# About
C# bindings for the GPUOpen Vulkan Memory Allocator library

Vulkan bindings use OpenTK 5 (preview)

Documentation can be found at <https://github.com/GPUOpen-LibrariesAndSDKs/VulkanMemoryAllocator>
